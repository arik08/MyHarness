# 매출 수주점검

## 목적

매출과 수주 상황을 목표 대비 관점에서 정리하고 회복 액션을 제안합니다.

## 사용 원칙

- 수치가 주어진 경우 목표 대비 차이, 전년 대비 차이, 월별 추세를 우선 정리합니다.
- 수주 가능성은 제공된 단계와 근거가 있을 때만 판단합니다.
- 고객명이나 계약 조건이 민감하면 비식별 요약을 사용합니다.

# 매출·수주 점검

|---|---:|---:|---:|---|

## 파이프라인 점검

## 회복 액션
