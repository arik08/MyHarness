# 세무

## 목적

세무 이슈와 신고·검토 필요사항을 경영진 보고용으로 정리합니다.

## 사용 원칙

- 최신 법령과 유권해석 확인은 `[세무 전문가 확인 필요]`로 표시합니다.
- 세목, 과세 기간, 영향 금액, 리스크 수준을 분리합니다.
- 확정된 세무 판단처럼 표현하지 않습니다.

## 출력 형식

```md
# 세무 이슈 점검

## 핵심 리스크

## 세목별 현황
| 세목 | 이슈 | 예상 영향 | 기한 | 조치 |
|---|---|---:|---|---|

## 전문가 확인 필요
```
