# 내부회계

## 목적

내부회계 통제 이슈와 개선 계획을 정리합니다.

## 사용 원칙

- 통제 설계, 운영, 증빙, 결함, 개선 조치를 분리합니다.
- 중요 결함 가능성이 있으면 `[감사 대응 필요]`로 표시합니다.
- 실제 감사 시스템이나 증빙 저장소 조회는 수행하지 않습니다.
