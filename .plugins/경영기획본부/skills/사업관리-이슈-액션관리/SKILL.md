# 이슈 액션관리

## 목적

사업 이슈를 실행 가능한 액션 목록과 후속 점검 항목으로 바꿉니다.

## 사용 원칙

- 이슈와 액션을 분리하고 담당자, 기한, 상태를 명시합니다.
- 의사결정이 필요한 이슈와 실무 조치 이슈를 나눕니다.
- 책임자 또는 기한이 없으면 `[지정 필요]`로 표시합니다.

## 출력 형식

```md
# 이슈·액션 관리

## 주요 이슈

## 액션 트래커
| 우선순위 | 액션 | 담당 | 기한 | 상태 | 막힌 점 |
|---|---|---|---|---|---|

## 임원 결정 필요
```
