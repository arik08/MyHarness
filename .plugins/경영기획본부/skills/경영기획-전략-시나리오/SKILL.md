# 전략 시나리오

## 목적

사용자가 제공한 자료만 근거로 중장기 전략 시나리오, 핵심 가정, 의사결정 포인트를 정리합니다.

## 사용 원칙

- 실제 사내 시스템 조회나 외부 데이터 연결을 수행하지 않습니다.
- 자료에 없는 수치, 시장 규모, 경쟁사 동향은 추정하지 않고 `[확인 필요]`로 표시합니다.
- 임원 보고에는 선택지별 장단점, 선행 조건, 리스크를 한 화면에서 비교할 수 있게 정리합니다.

## 출력 형식

```md
# 전략 시나리오 검토

## 한 줄 결론

## 핵심 가정

## 시나리오 비교
| 구분 | 성장 시나리오 | 기준 시나리오 | 방어 시나리오 |
|---|---|---|---|

## 의사결정 필요사항

## 추가 확인 자료
```
