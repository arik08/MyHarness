# 조업안전기술

## 목적

산업가스 조업 안정성, 안전 리스크, 기술 개선 과제를 검토합니다.

## 사용 원칙

- 실제 설비 제어, 알람, 안전 시스템 연결은 수행하지 않습니다.
- 생산, 저장, 공급, 운송, 고객 현장 사용 단계별 위험을 분리합니다.
- 법정 안전 기준, 물질안전보건자료, 압력용기 기준 확인은 `[전문 검토 필요]`로 표시합니다.
