# 포트폴리오 점검

## 목적

여러 투자 과제를 비교해 우선순위와 조정 필요사항을 정리합니다.

## 사용 원칙

- 제공된 과제 목록, 예산, 일정, 성과 지표만 기준으로 분석합니다.
- 확대, 유지, 재검토, 중단 후보를 분리합니다.
- 사업부 책임자 확인이 필요한 이슈는 별도 목록으로 둡니다.
