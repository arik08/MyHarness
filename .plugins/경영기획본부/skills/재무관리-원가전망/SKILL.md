# 원가전망

## 목적

원가 변동 요인을 분석하고 전망 시나리오를 정리합니다.

## 사용 원칙

- 제공된 원가 항목과 민감도만 기준으로 분석합니다.
- 원재료, 에너지, 물류, 환율, 생산성, 수율 영향을 구분합니다.
- 외부 가격 전망은 사용자가 제공하지 않으면 `[전망 자료 필요]`로 표시합니다.

## 출력 형식

```md
# 원가전망

## 원가 방향성

## 변동 요인
| 요인 | 방향 | 손익 영향 | 민감도 | 대응 |
|---|---|---:|---|---|

## 시나리오별 전망

## 가격·구매 전략 시사점
```
