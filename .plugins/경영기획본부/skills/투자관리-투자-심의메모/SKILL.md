# 투자 심의메모

## 목적

투자 안건을 임원 심의에 필요한 판단 항목으로 재구성합니다.

## 사용 원칙

- 투자금, 회수기간, NPV, IRR, 전략 적합성 자료가 없으면 `[보완 필요]`로 표시합니다.
- 찬반 결론보다 승인 조건과 추가 확인사항을 명확히 분리합니다.
- 실제 투자관리시스템 조회는 수행하지 않습니다.

## 출력 형식

```md
# 투자 심의메모

## 안건 요약

## 투자 판단
| 항목 | 내용 | 판단 | 확인 필요 |
|---|---|---|---|

## 주요 리스크

## 승인 조건

## 다음 단계
```
