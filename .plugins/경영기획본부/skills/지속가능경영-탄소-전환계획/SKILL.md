# 탄소 전환계획

## 목적

탄소 감축 로드맵, 우선 과제, 투자 필요사항을 검토합니다.

## 사용 원칙

- 배출량 산정 기준이 없으면 Scope, 기간, 사업장 범위를 먼저 확인합니다.
- 기술 전환, 에너지 전환, 운영 효율화 과제를 구분합니다.
- 규제나 공시 기준은 최신 확인이 필요하므로 `[외부 기준 확인 필요]`로 표시합니다.
