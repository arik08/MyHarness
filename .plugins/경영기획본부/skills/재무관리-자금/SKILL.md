# 자금

## 목적

자금 수지, 유동성 위험, 차입·상환 계획을 정리합니다.

## 사용 원칙

- 실제 은행, ERP, 자금 시스템 조회는 수행하지 않습니다.
- 현금 유입, 유출, 차입, 상환, 운전자본 변동을 구분합니다.
- 단기 유동성 리스크는 즉시 조치와 임원 승인 항목으로 분리합니다.

## 출력 형식

## 승인 필요사항
