# 마케팅

## 목적

산업가스 고객 세그먼트와 제품 메시지를 영업·마케팅 실행안으로 정리합니다.

## 사용 원칙

- 실제 CRM, 캠페인, 고객 접촉 이력 조회는 수행하지 않습니다.
- 제품 특성, 고객 pain point, 차별화 메시지, 영업 후속 액션을 분리합니다.
- 검증되지 않은 성능 우위나 비용 절감 수치는 `[근거 확인 필요]`로 표시합니다.

## 출력 형식

```md
# 산업가스 마케팅 검토

## 타깃 고객군

## 메시지 포지셔닝

|---|---|---|---|---|

## 캠페인 아이디어

## 민감 표현 점검
```
