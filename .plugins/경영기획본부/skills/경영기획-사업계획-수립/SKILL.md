# 사업계획 수립

## 목적

사업부 또는 전사 단위의 연간 사업계획 초안을 구조화합니다.

## 사용 원칙

- 사용자가 제공한 KPI, 예산, 조직, 과제 정보를 우선합니다.
- 목표와 실행과제가 분리되지 않으면 `목표`, `핵심 과제`, `성과 지표`, `필요 지원`으로 재분류합니다.
- 실행 주체, 일정, 지표가 비어 있으면 `[보완 필요]`로 표시합니다.

## 출력 형식

```md
# 사업계획 초안

## 목표 요약

## 핵심 KPI
| KPI | 목표 | 전년 대비 | 담당 | 확인 필요 |
|---|---:|---:|---|---|

## 중점 추진 과제

## 예산·인력 요청

## 임원 의사결정 사항
```
