# 이해관계자 공시

## 목적

지속가능경영 관련 대외 메시지, 예상 질문, 답변 방향을 정리합니다.

## 사용 원칙

- 제공되지 않은 약속, 목표, 대외 선언을 새로 만들지 않습니다.
- 이해관계자별 관심사를 분리해 메시지를 조정합니다.
- 법정 공시 또는 인증 문구는 `[전문 검토 필요]`로 표시합니다.

|---|---|---|---|

## 민감 표현 점검

## 후속 확인 자료
