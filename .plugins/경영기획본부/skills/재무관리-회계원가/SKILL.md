# 회계원가

## 목적

회계·원가 이슈를 결산과 손익 영향 중심으로 정리합니다.

## 사용 원칙

- 회계 기준 판단은 최종 자문이 아니라 검토 초안으로 표시합니다.
- 원가 배부 기준, 일회성 비용, 재고, 감가상각, 충당부채를 구분합니다.
- 회계 정책 확인이 필요한 항목은 `[회계 검토 필요]`로 표시합니다.

## 출력 형식

```md
# 회계원가 검토

## 결산 영향 요약

## 주요 이슈
| 이슈 | 손익 영향 | 회계 처리 | 원가 영향 | 확인 필요 |
|---|---:|---|---:|---|

## 후속 조치
```
