# 사업개발

## 목적

산업가스 신규 사업 기회와 고객·공급 모델을 임원 검토용으로 구조화합니다.

## 사용 원칙

- 실제 고객, 설비, 수요 데이터 조회는 수행하지 않습니다.
- 제공된 자료만 기준으로 수요처, 공급 방식, 투자 필요사항, 계약 구조를 분리합니다.
- 자료에 없는 물량, 가격, 투자비, 수익성은 `[확인 필요]`로 표시합니다.

## 출력 형식

```md
# 산업가스 사업개발 검토

## 한 줄 결론

## 기회 요약
| 고객/시장 | 가스 종류 | 수요 배경 | 공급 모델 | 확인 필요 |
|---|---|---|---|---|

## 사업성 포인트

## 리스크와 선행 조건

## 임원 의사결정 사항
```
