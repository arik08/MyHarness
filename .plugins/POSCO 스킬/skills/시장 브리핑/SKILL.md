# 시장 브리핑

이 스킬은 시장 동향 자료를 짧은 회의 브리핑 형태로 정리할 때 사용합니다.

## 사용 원칙

- 사용자가 제공한 기사, 표, 메모, 대화 내용을 기준으로 정리합니다.
- 출처가 불명확한 시장 전망과 가격 정보는 `[확인 필요]`로 표시합니다.
- 고객, 산업, 경쟁사 관점의 시사점을 분리합니다.

## 출력 구조

- 오늘의 시장 한 줄
- 수요/가격 동향
- 고객 및 경쟁사 움직임
- 리스크와 기회
- 후속 확인 항목
