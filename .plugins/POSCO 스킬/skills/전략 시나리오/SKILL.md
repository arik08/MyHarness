# 전략 시나리오

이 스킬은 전략 시나리오 초안을 빠르게 구성할 때 사용합니다.

## 사용 원칙

- 사용자가 제공한 자료와 대화 맥락만 근거로 사용합니다.
- 확인되지 않은 수치, 계획, 조직 정보는 `[확인 필요]`로 표시합니다.
- 낙관, 기준, 보수 시나리오를 나누고 각 시나리오의 전제와 대응 방향을 함께 정리합니다.

## 출력 구조

- 핵심 요약
- 주요 전제
- 시나리오별 영향
- 의사결정 포인트
- 추가 확인 필요 자료
