# 원료 가격 동향

이 스킬은 원료 가격과 공급 이슈를 구매 의사결정용 메모로 정리할 때 사용합니다.

## 사용 원칙

- 가격, 물량, 공급처 정보는 제공 자료에 있는 범위에서만 사용합니다.
- 확인되지 않은 가격 전망과 계약 조건은 `[확인 필요]`로 표시합니다.
- 단기 대응과 중기 조달 전략을 구분합니다.

## 출력 구조

- 가격 동향 요약
- 변동 요인
- 공급 리스크
- 협상/조달 포인트
- 확인 필요 데이터
