# 조업 이슈 요약

이 스킬은 현장 조업 이슈를 생산 회의 메모 형태로 정리할 때 사용합니다.

## 사용 원칙

- 실적, 설비, 품질, 안전 정보는 사용자가 제공한 내용만 사용합니다.
- 원인 분석이 확정되지 않은 항목은 `[확인 필요]`로 표시합니다.
- 조업 영향, 대응 현황, 후속 조치를 구분합니다.

## 출력 구조

- 조업 현황 요약
- 주요 이슈
- 영향 범위
- 대응 및 후속 조치
- 확인 필요 자료
