# 기술동향 리서치

이 스킬은 기술 동향 자료를 연구 과제 검토용으로 정리할 때 사용합니다.

## 사용 원칙

- 논문, 기사, 특허, 내부 메모 등 사용자가 제공한 자료를 기준으로 정리합니다.
- 기술 성숙도, 적용 가능성, 성능 수치는 근거가 없으면 `[확인 필요]`로 표시합니다.
- 기술 설명, 적용 가능성, 리스크, 후속 실험 아이디어를 분리합니다.

## 출력 구조

- 기술 개요
- 동향 요약
- POSCO 업무 적용 가능성
- 기술/사업 리스크
- 후속 검토 과제
