# OpenHarness 패치 노트

이 문서는 로컬 Git 커밋 기록을 바탕으로, 비개발자도 이해하기 쉽게 날짜별 주요 개선사항만 정리한 것입니다.
테스트 보강, 문서의 작은 문구 수정, 내부 정리 같은 자잘한 변경은 관련 기능 아래에 묶었습니다.

표기 기준:

- `원본 OpenHarness`: upstream OpenHarness 또는 다른 contributor의 커밋입니다.
- `Myeongcheol`: Git author가 `Myeongcheol` 또는 `Myeongcheol_Oh`인 커밋입니다.

## 2026-04-25 (Myeongcheol)

- 모델이 답변을 만들 때 얼마나 깊게 추론할지 조절하는 옵션을 추가했습니다. 단순한 작업은 가볍게, 복잡한 작업은 더 깊게 생각하도록 선택할 수 있는 기반이 생겼습니다.
- OpenAI 계열 모델의 기본 설정을 최신 사용 흐름에 맞게 정리했습니다. 사용자가 모델을 고를 때 예전 기본값 때문에 엉뚱한 모델이나 옵션으로 흐르는 일을 줄이는 목적입니다.

- 기존에 Git이 추적하던 UI 점검 로그와 임시 브라우저 파일을 인덱스에서 제거했습니다. 파일이 실제 작업 폴더에 남아 있더라도, 앞으로 커밋 대상에는 올라오지 않게 했습니다.
- 작업 지침에도 UI 점검 산출물 처리 기준을 추가했습니다. 앞으로 화면 점검 로그와 스크린샷은 루트에 흩뿌리지 않고, 무시되는 로컬 작업 폴더에 모으도록 기준을 세웠습니다.
