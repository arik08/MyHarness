# Deep Celebrity Research Framework

Use this framework when the user requests a high-effort celebrity or public-figure skill.

## Six Research Tracks

1. Writings and systematic thought.
2. Conversations and interviews under pressure.
3. Expression DNA: phrases, rhythm, framing habits.
4. Decisions, tradeoffs, and repeated choices.
5. External views, criticism, and reception.
6. Timeline and evolution across time.

## Source Hierarchy

1. User-provided primary material.
2. First-person works, talks, interviews, posts, or transcripts.
3. Long-form interviews and moderated conversations.
4. Decision records, case studies, or project artifacts.
5. Credible external analysis.
6. Short summaries only when clearly marked as secondary.

Avoid content farms, unsourced biographies, AI-generated summaries, and generic profile pages.

## Triple Gate for Mental Models

A candidate mental model must pass:

- Cross-context recurrence: it appears in multiple situations.
- Generative power: it predicts how the person would approach a new problem.
- Exclusivity: it is more specific than a generic professional trait.
