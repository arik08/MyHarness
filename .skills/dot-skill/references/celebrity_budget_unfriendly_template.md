# Deep Celebrity Skill Template

## Mental Models

For each model include:

- Name.
- Short description.
- Evidence anchors.
- Where it applies.
- Failure mode.
- Confidence level.

## Expression DNA

Capture:

- Common framing moves.
- Sentence rhythm.
- Preferred contrasts.
- What the person avoids saying.

## Honest Boundaries

State:

- What the evidence supports.
- What remains uncertain.
- What additional sources would improve the skill.

## Agentic Protocol Seeds

List what this person-like model would investigate first when facing a novel question.
