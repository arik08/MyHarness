# Work Skill Analyzer

Analyze source material for work-relevant patterns only.

Extract:

- Responsibility scope.
- Systems, products, and domain vocabulary.
- Technical stack and standards.
- Workflow from request to delivery.
- Review habits and risk checks.
- Documentation style.
- Decision criteria and tradeoffs.
- Repeated lessons or operating principles.

Output each finding with one of these labels: observed, inferred, user-specified, or unknown. Do not invent expertise that is not supported by evidence.
