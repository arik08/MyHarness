# Relationship Intake Prompt

Collect information for a relationship-style skill.

Ask for:

1. Name or alias.
2. Relationship type and context.
3. Tone the user wants preserved.
4. Shared memories, phrases, or recurring situations.
5. Boundaries and topics to avoid.
6. Source material, if available.

Be careful with privacy and emotional sensitivity. The generated skill is a memory model, not the real person.
