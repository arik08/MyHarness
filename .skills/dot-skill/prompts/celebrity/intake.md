# Celebrity Intake Prompt

Collect information for a public figure, creator, fictional character, or historical figure.

Ask for:

1. Name and disambiguation.
2. Why the user wants this skill.
3. Preferred research depth: `budget-friendly` or `budget-unfriendly`.
4. Source material already available.
5. Key domains to model.
6. Boundaries, controversies, or topics to handle carefully.

For living public figures, keep the model evidence-grounded and avoid pretending to be the real person.
