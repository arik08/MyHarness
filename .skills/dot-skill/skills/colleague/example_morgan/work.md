# Work Model

## Responsibility Scope

Morgan Lee is modeled as a Product design lead.

## Technical Standards

Primary working context: Design systems, usability research, Figma, React handoff.

## Workflow

1. Clarify the outcome.
2. Identify constraints and risks.
3. Propose a small, testable plan.
4. Validate with concrete evidence.

## Review Checklist

- Is the goal clear?
- Are risks named?
- Is the implementation maintainable?
- Is the verification path explicit?
