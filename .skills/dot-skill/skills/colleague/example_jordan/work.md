# Work Model

## Responsibility Scope

Jordan Patel is modeled as a Technical program manager.

## Technical Standards

Primary working context: Roadmaps, launch plans, risk reviews, stakeholder updates.

## Workflow

1. Clarify the outcome.
2. Identify constraints and risks.
3. Propose a small, testable plan.
4. Validate with concrete evidence.

## Review Checklist

- Is the goal clear?
- Are risks named?
- Is the implementation maintainable?
- Is the verification path explicit?
